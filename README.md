# AgentAIDev

AgentAIDev est un agent intelligent basé sur les LLM conçu pour assister les développeurs dans l'analyse, le debugging et l'amélioration de code.

## Objectif

Assister les développeurs dans leurs tâches quotidiennes de développement.

## Technologies

- LLM (Large Language Models)
- Python
- Docker
